# Vocal Suite Ultra Pro - Beta Test

## Tester Steps

1. Install the VST3 package.
2. Open your DAW.
3. Load Vocal Suite Ultra Pro on a vocal track.
4. Test factory presets.
5. Test bypass and automation.
6. Save and reopen the project.
7. Report any crash, noise, silence, high CPU, or UI issue.

## Required Report

- OS version
- DAW name and version
- Audio interface
- Sample rate
- Buffer size
- Plugin format
- Issue description
- Steps to reproduce
- Screenshot or video if available

## Critical Issues

Report immediately:

- DAW crash
- Plugin scan failure
- Loud pop/noise
- Output silence
- Preset load failure
- License state reset
- Installer failure
