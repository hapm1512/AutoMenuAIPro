# Vocal Suite Ultra Pro — Quick Start

## 1. Insert Plugin
Add Vocal Suite Ultra Pro to your vocal track.

## 2. Set Input
Sing the loudest part.
Adjust input so peaks stay below -6 dBFS.

## 3. Choose Preset
Start with one preset:

- Clean Vocal
- Modern Pop
- Live Clean
- Podcast Voice
- Broadcast FM

## 4. Adjust Gate
Raise gate only until background noise lowers.
Do not cut vocal endings.

## 5. Adjust EQ
Use small changes.
Cut mud before boosting highs.

## 6. Adjust De-Esser
Reduce harsh S sounds.
Avoid dulling the whole vocal.

## 7. Adjust Compressor
Aim for stable vocal volume.
Avoid excessive pumping.

## 8. Add Color
Use Analog Color lightly.
Increase only if more character is needed.

## 9. Set Output
Set ceiling below 0 dBFS.
Match bypass and processed loudness.

## 10. Save Preset
Save your adjusted setting as a user preset.
