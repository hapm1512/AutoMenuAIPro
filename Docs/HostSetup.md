# Vocal Suite Ultra Pro — Host Setup Guide

## Cubase
1. Copy VST3 to the VST3 folder.
2. Open Cubase.
3. Rescan plugins if needed.
4. Insert plugin on vocal audio track.
5. Test bypass, preset loading, and automation.

## Reaper
1. Add VST3 path in Preferences.
2. Rescan plugins.
3. Insert plugin on vocal track.
4. Test latency and CPU.

## Studio One
1. Confirm VST3 folder is enabled.
2. Rescan plugins.
3. Insert plugin on vocal track.
4. Save and reload session.

## Ableton Live
1. Enable VST3 in preferences.
2. Rescan plugins.
3. Load plugin on audio track.
4. Test automation and presets.

## FL Studio
1. Add VST3 path in Plugin Manager.
2. Scan installed plugins.
3. Mark as favorite if needed.
4. Insert plugin in mixer slot.

## Recommended Test
For every host:

- Load plugin.
- Change presets.
- Save project.
- Close host.
- Reopen project.
- Confirm settings restore.
