# Support Policy

Product: Vocal Suite Ultra Pro
Company: Hai Pham Audio

## Supported

- Installation issues.
- License activation issues.
- VST3 scanning issues.
- Preset loading issues.
- Reproducible crashes.

## Required Support Information

- DAW name and version.
- Windows version.
- Audio interface.
- Sample rate.
- Buffer size.
- Screenshot or crash report.

## Initial Supported Hosts

- Cubase.
- Reaper.
- Studio One.
- Ableton Live.
- FL Studio.
