# Vocal Suite Ultra Pro — Troubleshooting

## No Sound
Check:

- Track is not muted.
- Plugin is not bypassed.
- Input/output routing is correct.
- Output trim is not too low.
- DAW audio engine is enabled.

## Vocal Sounds Distorted
Check:

- Input level is not clipping.
- Analog drive is not too high.
- Saturation is not excessive.
- Output ceiling is not too high.
- DAW track is not clipping after plugin.

## Gate Cuts Words
Lower gate threshold.
Increase release or hold.
Use lighter gate settings.

## S Sounds Too Harsh
Increase de-esser amount.
Adjust frequency focus between 5 kHz and 8 kHz.
Use split-band mode for natural results.

## Vocal Sounds Dull
Reduce de-esser amount.
Reduce high-frequency cuts.
Add small air band boost.

## CPU Too High
Increase DAW buffer size.
Disable unused modules.
Reduce oversampling where available.
Freeze the track.

## Presets Do Not Load
Check preset folder path.
Restart the DAW.
Re-scan plugin if needed.
Verify the preset file is not corrupted.

## Plugin Not Found In DAW
Check VST3 install path.
Rescan plugins in DAW.
Restart DAW.
Make sure the plugin format matches the DAW.

## Automation Issues
Use smooth automation curves.
Avoid very fast automation on heavy DSP parameters.
Verify the parameter is exposed to the DAW.
