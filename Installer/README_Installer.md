# Vocal Suite Ultra Pro Installer

## Recommended install

Use the user-level installer first:

```powershell
powershell -ExecutionPolicy Bypass -File .\Install_VST3_User.ps1
```

## System install

Run PowerShell as Administrator:

```powershell
powershell -ExecutionPolicy Bypass -File .\Install_VST3_System.ps1
```

## Uninstall

```powershell
powershell -ExecutionPolicy Bypass -File .\Uninstall_VST3.ps1
```

## DAW Scan

After installation, rescan VST3 plug-ins in the DAW.
